# Project Porfolio
```sh
git clone https://github.com/mrgomezsv/porfolio_mr.git
cd porfolio_mr
python -m venv env
.\env\Scripts\Activate.ps1
pip3 install -r requirements.txt
$Env:FLASK_APP = "main.py"
flask run
```
